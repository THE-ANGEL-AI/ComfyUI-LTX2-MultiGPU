# Documentation (docs/)

> Эта папка — **deep-docs** проекта. Для быстрого старта и install-инструкций — [**README.md в корне**](../README.md) (landing-page).

* [**CHANGELOG.md**](./CHANGELOG.md) — история изменений по версиям (Keep-a-Changelog формат, semver).
* [**SECURITY.md**](./SECURITY.md) — полиция безопасности: как сообщить об уязвимости, supported-versions, response-timeline, GPL-3.0 copyleft obligations в downstream-форках.
* [**CONTRIBUTING.md**](./CONTRIBUTING.md) — гайд для контрибьюторов: fork → branch → PR, Conventional Commits, DCO sign-off, code-style.

> Все три файла распознаются GitHub'ом автоматически: `SECURITY.md` отображается в **Security** tab, `CONTRIBUTING.md` показывается при создании Pull Request. См. <https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file>.

`docs/index.md` распространяется под **GPL-3.0-or-later** (как и код проекта).
